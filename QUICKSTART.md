# 🚀 快速开始（极简版）

## 第一次使用（只需做一次）

### 1️⃣ 安装 Python

- 访问：https://www.python.org/downloads/
- 点击 **"Download Python 3.11.x"**
- 运行下载的文件
- **勾选** "Add Python to PATH"
- 点击 **Install Now**

### 2️⃣ 安装本程序

- 下载本程序 ZIP 文件
- 解压到任意文件夹
- 进入 `standalone_app` 文件夹
- **双击 `install.bat`**
- 等待安装完成

## 日常使用

### 启动程序

**双击 `run.bat`**

等待几秒，浏览器会自动打开。

如果没有自动打开，手动访问：`http://localhost:8501`

### 关闭程序

- 关闭浏览器标签页
- 关闭黑色的命令行窗口

## 如果启动失败

1. **双击 `install.bat`** 重新安装依赖
2. 查看 `TROUBLESHOOTING.md` 故障排除指南
3. 在 GitHub 提交 Issue 寻求帮助

## 文件说明

| 文件 | 作用 |
|------|------|
| `install.bat` | 安装依赖（首次使用运行） |
| `run.bat` | 启动程序（日常使用） |
| `run.ps1` | PowerShell 版本的启动脚本（备用） |
| `data/` | 数据文件夹（自动创建） |

---

**💡 提示**：把 `run.bat` 发送到桌面快捷方式，以后双击桌面图标就能启动！
